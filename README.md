# Sales Performance Intelligence Dashboard

Project Overview

The Sales Performance Intelligence Dashboard is a data analytics project developed using Power BI, SQL, Python, and Excel to analyze business sales performance and generate KPI-driven insights.

This project helps in understanding revenue trends, customer behavior, product performance, region-wise sales, and monthly business growth for better decision-making.

 Project Objective

The main objective of this project is to transform raw sales data into meaningful business insights using data cleaning, SQL analysis, and interactive dashboard visualization.

The dashboard supports:
- Revenue tracking
- Profit monitoring
- Customer analysis
- Product performance evaluation
- Region-wise sales comparison
- Monthly sales growth analysis

 Dataset Information

The dataset includes structured sales records with the following fields:

- Order ID
- Customer Name
- Product
- Category
- Region
- Sales
- Profit
- Quantity
- Order Date
- Month
- Year
- Customer Type
- Payment Mode

The dataset was created using Excel and saved in CSV format.

 Python Data Cleaning Process

Data cleaning was performed using Python libraries Pandas and NumPy.

Main preprocessing steps:

- Removed duplicate records
- Handled missing values
- Converted date columns into proper format
- Created month-wise analysis columns
- Saved cleaned dataset for SQL and Power BI usage

This improved data quality and reporting accuracy.


 SQL Analysis

SQL was used for business insight generation.

Queries performed include:

- Total Revenue Calculation
- Top-Selling Products
- Region-wise Sales Analysis
- Monthly Sales Performance
- Customer Type Analysis
- Profit Distribution

These queries helped identify business performance trends.

 Key Business Insights

Major findings from the project:

- Electronics generated the highest revenue
- West region showed the strongest sales performance
- Premium customers contributed higher revenue
- UPI and Card were the most preferred payment modes
- Monthly trends helped identify peak sales periods

These insights support better business planning and strategic decision-making.
